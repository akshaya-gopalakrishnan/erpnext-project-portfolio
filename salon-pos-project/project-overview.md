# Salon POS System with Employee Commission

## System Overview

Standard POS systems are designed for retail businesses.

Salon businesses require additional functionality including:

• Employee service tracking  
• Commission calculation  
• Shift cash management

This project extends ERPNext POS to support these requirements.

---

## Core Features

• Employee tagging per service  
• Opening and closing shift management  
• Employee productivity tracking  
• Automated commission calculation
