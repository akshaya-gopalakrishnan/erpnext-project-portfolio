# Typing Center Management System (TCMS)

## System Overview

Typing centers process large numbers of client applications such as visa processing, document typing, and government services.

Standard ERP systems do not provide a workflow for tracking these applications.

The TCMS system was built as a custom Frappe application to manage the full lifecycle of client applications.

---

## Business Problem

Typing centers need to manage:

• Client records  
• Multiple services per client  
• Document expiry tracking  
• Application status monitoring  
• Payment tracking

Manual tracking using spreadsheets leads to data loss and poor visibility.

---

## Solution

The TCMS application centralizes all operations into a structured ERP system.

Key capabilities include:

• Client management  
• Application lifecycle tracking  
• Document expiry alerts  
• Payment tracking dashboard  
• Client portal for application status
