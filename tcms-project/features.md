# TCMS Features

## Client Management

A custom Client DocType was created to store client information including:

• Name  
• Contact details  
• Client type  
• Service history

---

## Document Management

Clients often submit documents such as passports or visas.

The system tracks:

• Document type  
• File attachment  
• Expiry date

Expiry dates are monitored through automated background scripts.

---

## Expiry Alert System

A scheduled script runs daily and checks for documents approaching expiry.

If documents are nearing expiry:

• Alerts are generated  
• Users are notified through ERP notifications

This prevents missed renewals.

---

## Financial Dashboard

A custom financial dashboard displays application payment status visually.

Statuses include:

• Unpaid  
• Paid  
• Overdue

This helps staff quickly identify pending payments.

---

## Client Portal

A mobile-friendly portal allows clients to track application progress.

Clients can view:

• Application status  
• Submitted documents  
• Service updates
